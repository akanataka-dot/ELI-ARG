# ELI ― スタートガイド(プレイヤー用・ネタバレなし)

所要時間: 約2時間 / 1人用 / 必要なもの: ブラウザとメモ

## あなたの役割

あなたは小学生の子を持つ親です。リビングに置きっぱなしになっていた、学校配布の学習タブレットに、見慣れないAIアプリ「ELI」を見つけました。

## 遊び方 ― 2つの窓を開く

このゲームは、ブラウザの**ウィンドウ(タブ)を2つ**使って遊びます。

1. **`index.html`** ― 子供の学習タブレットの画面です。ここから物語が始まります。
2. **`search.html`** ― 検索サイト(LUMO検索)です。**外の世界**への入り口です。タブレットのホームにある「調べる(検索)」からも、新しいタブで開けます。

タブレットの中で見かけた「気になる言葉・番号・名前・会社名」は、なんでも LUMO で検索してください。タブレットの中のAIは、すべてを話してはくれません。**真実は、外の世界に散らばっています。**

## 行き詰まったら

- 直近で目にした固有名詞や番号を、検索していないものがないか確認する
- 一度読んだページも、新しい事実を知ってから読み返すと違って見えます
- 紙のメモを取ることを強くおすすめします(日付・番号・人名)

## 注意

- **PCでのプレイを推奨します**(ウィンドウを2つ並べるため。スマホでは快適に遊べません)
- ブラウザの「戻る」ではなく、ページ内のリンクやアドレス欄を使う場面があります
- 最初からやり直したいときは、エンディング画面の「最初から見る」か、タブレットの「保護者用設定」最下部の「端末データを初期化する」を押してください
- エンディングは1つではありません
- 音は出ません。夜、一人で遊ぶことを推奨します
